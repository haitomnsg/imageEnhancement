**[Lab 2 Manual]{.underline}**

Course: **AICL 311 Image Processing**

Lab Title: **Image Enhancement using Point Processing and Histogram
Techniques**

**Objectives**

-   To implement basic point processing techniques for image
    enhancement.

-   To apply intensity transformations using Python on grayscale images.

-   To analyze the effects of different intensity transformations on
    images.

-   To perform histogram-based enhancement techniques, including
    equalization.

-   To apply contrast stretching and observe visibility improvement.

-   To implement binary thresholding and compare results with original
    images.

**[Lab Instructions]{.underline}**

**1. Setup**

-   Use Jupyter Notebook or Google Colab for implementation.

**2. Image Input**

-   Load at least 2--3 grayscale images of different characteristics
    (dark, bright, low contrast).

-   Convert color images to grayscale if required.

**3. Implementation Tasks**

Students must implement the following techniques in Python:

A. Point Processing Techniques

-   Image Negative

-   Log Transformation

-   Inverse Log Transformation

-   Gamma Correction

-   Contrast Stretching

-   Binary Thresholding

B. Histogram Processing

-   Compute histogram of original image

-   Apply histogram equalization

-   Compare original and enhanced histograms

**4.** **Observation**

For each technique:

-   Display original and processed images side by side

-   Observe changes in brightness, contrast, and visibility

-   Comment briefly on the effect of each transformation

**5. Analysis**

-   Compare which method improves image visibility the most

-   Identify which techniques work best for dark or low-contrast images

-   Discuss limitations of thresholding and gamma correction

Students must submit a single Jupyter Notebook (.ipynb) containing:

-   Lab title

**3. Output Results**
